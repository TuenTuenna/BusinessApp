# BusinessApp
## simple business ios app

- tapbar controller
- table view
- collection view

#### used frameworks

- MapKit: to draw map
- Accounts: to check if user can send email
- MessageUi: to send email

#### features

- shows information of a company
- displays images in collection view
- makes a phone call
- sends an email
- shows directions to a company
